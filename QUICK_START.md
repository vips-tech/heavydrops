## 🔧 Troubleshooting

### Website Not Loading?
1. Check if server is running: `http://localhost:5001`
2. Look for errors in terminal
3. Restart server: `python app.py`

### Images Not Showing?
- Images are loaded from Unsplash CDN
- Check internet connection
- Fallback placeholders will show if images fail

### Styles Look Wrong?
1. Clear browser cache (Ctrl+Shift+R)
2. Check CSS files are loading
3. Verify server is serving static files
