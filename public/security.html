How we protect your data and ensure a safe jewelry discovery experience

Our Security Commitment

Heavy Drops employs enterprise-grade security measures to protect your personal information, financial data, and platform interactions. Your trust is our priority.

1. Data Encryption

In Transit

All data transmitted between your device and our servers is encrypted using:

  • TLS 1.3: Latest transport layer security protocol
  • 256-bit SSL: Bank-grade encryption for all connections
  • HTTPS: Secure protocol for all pages and API calls

At Rest

Data stored on our servers is protected with:

  • AES-256 encryption for sensitive information
  • Encrypted database storage
  • Secure key management systems
  • Regular encryption audits

2. Payment Security

We implement multiple layers of payment protection:

✓ PCI DSS Compliant
✓ Secure Payment Gateway
✓ 3D Secure Authentication

Payment Processing

  • We never store complete credit card numbers
  • All payments processed through certified gateways
  • Two-factor authentication for high-value transactions
  • Real-time fraud detection and prevention
  • Tokenization of payment information

3. Account Security

Authentication

  • OTP Verification: SMS-based one-time passwords
  • Email Verification: Secure email confirmation
  • Session Management: Automatic timeout after inactivity
  • Device Recognition: Alerts for new device logins

Password Security

  • Passwords hashed using bcrypt algorithm
  • Minimum password strength requirements
  • Secure password reset process
  • Protection against brute force attacks

4. Infrastructure Security

Server Protection

  • Cloud infrastructure with enterprise security
  • Firewall protection and intrusion detection
  • Regular security patches and updates
  • DDoS protection and mitigation
  • 24/7 server monitoring

Database Security

  • Encrypted database connections
  • Role-based access controls
  • Regular automated backups
  • Disaster recovery procedures

5. Application Security

We follow secure development practices:

  • Code Reviews: All code reviewed for security vulnerabilities
  • Penetration Testing: Regular security assessments
  • Vulnerability Scanning: Automated security scans
  • Input Validation: Protection against injection attacks
  • CSRF Protection: Cross-site request forgery prevention
  • XSS Prevention: Cross-site scripting protection

6. Fraud Prevention

Detection Systems

  • Real-time transaction monitoring
  • Behavioral analysis and anomaly detection
  • IP address and device fingerprinting
  • Velocity checks for suspicious activity

User Protection

  • Transaction notifications via SMS and email
  • Spending limits and controls
  • Ability to freeze account instantly
  • Dispute resolution process

7. Privacy Controls

You control your data:

  • Granular privacy settings
  • Control over data sharing with sellers
  • Ability to delete account and data
  • Export your data anytime
  • Opt-out of marketing communications

8. Compliance & Certifications

We comply with industry standards and regulations:

  • Data Protection: Compliance with Indian data protection laws
  • Payment Security: PCI DSS Level 1 compliance
  • Security Standards: ISO 27001 aligned practices
  • Regular Audits: Third-party security assessments

9. Incident Response

In the unlikely event of a security incident:

  • Immediate investigation and containment
  • Notification to affected users within 72 hours
  • Coordination with law enforcement if necessary
  • Post-incident analysis and improvements
  • Transparent communication about the incident

10. Best Practices for Users

Help us keep your account secure:

  • Use a strong, unique password
  • Never share your password or OTP
  • Log out after using shared devices
  • Keep your contact information updated
  • Review account activity regularly
  • Report suspicious activity immediately
  • Enable all available security features

11. Report Security Issues

If you discover a security vulnerability:

  • Email: security@heavydrops.com
  • Response Time: Within 24 hours
  • Responsible Disclosure: We appreciate responsible reporting

Please do not publicly disclose security issues until we've had a chance to address them.
